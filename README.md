# Zerodha Stock Trading Platform

A Full Stack MERN application inspired by Zerodha for learning and practice purposes.

## Features
- User Authentication
- Portfolio Management
- Watchlist Functionality
- Responsive Dashboard UI
- REST API Integration

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JavaScript

## Purpose
This project was built to improve Full Stack Development skills and understand real-world trading platform workflows.
